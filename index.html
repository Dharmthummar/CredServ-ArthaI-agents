Phase 1 — KYC Extractor
VLM + Deterministic Verifier

Uses a local Vision-Language Model (Ollama llama3.2-vision) to extract structured JSON from Indian bank statements — clean, skewed, and bilingual. No regex. No brittle OCR.

✦ Universal JSON schema prompt
✦ Deterministic balance verification (Decimal-safe)
✦ Auto-retry with failure reason injection
✦ Dual backend: Ollama (local) or Gemini (free tier)
Phase 2 — Collections Orchestrator
LangGraph State Machine

Deterministic LangGraph graph managing a borrower's lifecycle from D-15 to D+3. SMS, email, push, and AI voice call orchestrated with strict bounded autonomy — bounded by 8 negative constraints.

✦ 6 deterministic states, conditional routing
✦ D+3 AI voice agent (ARIA) with strict guardrails
✦ Instant human hand-off on dispute
✦ Full audit log — run_id, contacts, transitions
